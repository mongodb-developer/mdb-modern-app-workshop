# Introduction to MongoDB with Eventer

Welcome to the MongoDB integration workshop! In this hands-on session, we'll build Eventer, an event management application, while exploring the powerful features of MongoDB.

## What We'll Build

Eventer is a web application that allows users to create, view, edit, and delete events. We'll use MongoDB as our database, leveraging its flexible document model and powerful query capabilities.

## What You'll Learn

- Setting up a MongoDB Atlas database
- Performing CRUD operations with MongoDB
- Implementing search functionality
- Using MongoDB's aggregation pipeline
- Advanced features like indexing and geospatial queries
- Deploying your app with MongoDB Atlas
